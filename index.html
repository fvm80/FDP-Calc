FDP Calculator

Version: 1.0.0

The FDP Calculator helps plan crew duties in line with Pecotox Air OM-A Flight Time Limitations (FTL). The tool has two main tabs: SIMPLE (quick calculations) and SCHEDULE (full timetable-based calculations).

SIMPLE tab — Quick FDP calculation

Use this tab for a fast FDP estimate using basic duty parameters.

Note: All times are shown in Local Time (LT) based on the crew's acclimatised timezone.

  • Start of Duty — reporting time for the duty.
  • No. of Planned Sectors — number of sectors planned for this duty.
  • Split Start — STA of the flight before the split (if a break ≥ 4 hours is planned).
  • Split End — STD of the flight after the split.

Results included:

  • Basic (Standard Crew) — maximum FDP for standard crew (OMA 7.8.1).
  • With Extension — maximum FDP with extension (OMA 7.8.3).
  • Split — maximum FDP with Split Duty extension (OMA 7.11).
  • Crew+ (3,4) — maximum FDP for augmented crews (OMA 7.8.4).
  • Latest Time ON BLOCK — latest allowed end-of-duty time (ON BLOCK of the final sector).

SCHEDULE tab — Full schedule calculation

Use this tab to calculate FDPs from a full flight schedule (multiple legs).

  1. Set Home Base Offset
    Difference between Home Base local time and UTC. Default: +4:00 (UAE).
  2. Enter Flight Schedule
    Insert STD and STA in UTC only. Add legs with + Add Leg, remove with X.

While entering the schedule the tool will show the timeline converted to Home Base Local Time under the + Add Leg button.

Important: only the break between the last two sectors can be considered for Split Duty, even if other breaks exist.

Results included:

  • FDP Records Table — formatted like Journey Log entries; shows times in both UTC and LT and the calculated FDP.
  • MAX FDP / Latest Time ON BLOCK Table — same FDP results as in SIMPLE, with Latest Time ON BLOCK displayed in UTC.
